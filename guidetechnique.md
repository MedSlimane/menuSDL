Guide Technique : Dimensions des Composantes Graphiques (2D)
Dans un projet de jeu vidéo, la précision des dimensions garantit que les images ne seront ni floues (pixélisées), ni trop lourdes.

1. L'Écran de Jeu 
La référence standard actuelle est le Full HD.

Résolution : 1920/1080 pixels.
Ratio : 16:9.
Résolution de sortie (DPI) : 72 DPI (le standard pour les écrans).
2. Le Logo du Jeu
Le logo doit être utilisable sur l'écran titre mais aussi sur des icônes plus petites.

Format de création : Environ 1000/600 pixels.
Format de fichier : PNG avec fond transparent (couche Alpha).
3. L'Interface Utilisateur (UI)
Les éléments de l'UI sont proportionnels à l'écran 1920/1080 pixels.

Boutons du menu : 400/100 pixels (pour les grands) ou 200/80 pixels (pour les petits).
Icônes du HUD (Vie, Munitions) : 64 /64 pixels ou 128/128 pixels.
Barre de santé (HP) : 300/40 pixels.
Zone de texte (Dialogue) : 1200/250 pixels (souvent placée en bas de l'écran).
4. Personnages et Sprites
La taille dépend de la place que prend le personnage à l'écran.

Petit personnage (style Arcade) : 64/64 pixels.
Personnage détaillé (Action/Aventure) : 256/256 pixels ou 512/512 pixels.
Ennemis / Boss : Peuvent aller jusqu'à 1024/1024 pixels.
5. Les Décors (Backgrounds)
Fond fixe : 1920/1080 pixels.
Fond pour scrolling (défilement) : La largeur doit être un multiple de 1920 (ex: 3840/1080 px pour un niveau qui défile sur deux écrans).
