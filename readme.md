## menu principal
-Le menu principal doit contenir un arrière-plan (arrière-plan 1) ainsi qu'une musique de
fond qui se lance automatiquement au démarrage.
-Un menu doit contenir 5 boutons : Jouer, options, score, histoire et quitter.
-Un menu doit contenir un texte. Exemple : nom de votre jeu ou de votre groupe ...
-si je clique sur le bouton “jouer” ou j'appuie sur la touche “j”, le “sous menu
sauvegarde et chargement” s'affiche.
-si je clique sur le bouton “option” ou j'appuie sur la touche “o”, le “sous menu option”
s'affiche.
-si je clique sur le bouton “Meilleurs scores” ou j'appuie sur la touche “m”, le “sous
menu meilleurs scores” s'affiche.
-si je clique sur le bouton Quitter ou j'appuie sur la touche “echap”, je quitte le jeu.
- Si on passe la souris au-dessus d'un bouton, le bouton devient sélectionné et un son
bref est joué.

## sous menu option
-Le sous menu option doit contenir un deuxième arrière-plan (arrière-plan 2) avec une
autre une musique de fond.
-Le sous menu option doit gérer le volume du son (exemple 2 boutons) et le mode
d’affichage (normale ou plein écran) avec 2 boutons.
- si je clique sur le bouton "Augmenter ou j’appuie sur la touche “+”, le niveau sonore
augmente.
- si je clique sur le bouton "Diminuer" ou j’appuie sur la touche “-”, le niveau sonore
diminue.
- si je clique sur le bouton "Plein écran", la fenêtre s'étend sur l'intégralité de l'écran et
si je clique sur le bouton "Mode normal", la fenêtre retrouve sa taille initiale.
-si je clique sur retour, je reviens au “menu principal”.
- Si on passe la souris au-dessus d'un bouton, le bouton devient sélectionné et un son
bref est joué.

## sous menu sauvegarde 
- le sous menu sauvegarde doit contenir le deuxième arrière-plan (arrière-plan 2) avec la
musique de fond. Il affiche le message “Voulez-vous sauvegarder votre jeu” et 2
boutons (oui et non)
- si je clique sur le bouton “Oui”, il affiche les deux boutons : “charger le jeu” et
“nouvelle partie”.
- si je clique sur le bouton “nouvelle partie” ou j’appuie sur la touche “n”, il affiche le
“sous menu joueur”.
- Si on passe la souris au-dessus d'un bouton, le bouton devient sélectionné et un son
bref est joué.

## sous menu player 

- le sous menu doit contenir le deuxième arrière-plan (arrière-plan 2) avec la musique de
fond. Il affiche 2 boutons (mono joueur et multi joueurs)
- Si je clique sur l’un des deux boutons, 4 autres boutons s’affichent afin de choisir
(Avatar1 ou Avatar2// input1 ou input2)
- Si je clique sur le bouton “valider” ou j’appuie sur la touche “Entrée”, le “sous menu
meilleurs scores” s’affiche.
- si je clique sur retour, je reviens au menu principal.
- Si on passe la souris au-dessus d'un bouton, le bouton devient sélectionné et un son
bref est joué.


## sous menu score 
- le sous menu meilleurs scores doit contenir un troisième arrière-plan (arrière-plan 3)
sans musique de fond. Il affiche un espace pour la saisie du nom du joueur à la fin du
jeu afin de sauvegarder le nom et le score du joueur.
- Si je clique sur le bouton “valider” ou j’appuie sur la touche “Entrée”, la liste des trois
meilleurs scores s'affiche accompagnée d'une musique de fond célébrant la victoire.
-si je clique sur retour, je reviens au menu principal.
- si j’appuie sur la touche ‘e’, le “sous menu énigme” s’affiche
-si je clique sur le bouton Quitter ou j'appuie sur la touche “echap”, je quitte le jeu.
- Si on passe la souris au-dessus d'un bouton, le bouton devient sélectionné et un son
bref est joué.

##  sous menu engime 
- le sous menu énigme doit contenir le quatrième arrière-plan (arrière-plan 4) sans
musique de fond. Il affiche deux boutons (Quiz et Puzzle)
- si je clique sur le bouton “Quiz”, il lance une musique de fond qui exprime une montée
de suspense (battement de tambour) et affiche le message Quiz, une bande sur laquelle
on affichera la question et 3 boutons pour les 3 propositions.
-si j'appuie sur la touche “echap”, je quitte le jeu.
